# Security Policy

This repository is an educational point-of-sale project for controlled cyber lab exercises.

## Reporting a Vulnerability

Please open a private security advisory or contact the maintainer directly with:
- clear reproduction steps
- affected files/endpoints
- potential impact

Do not post exploit details in public issues until a fix is available.

## Lab Credentials and Safety

- Seeded lab credentials are disabled by default.
- Lab users are only created when `SEED_LAB_USERS=true`.
- Any lab defaults are for training environments only and must not be used in production.
